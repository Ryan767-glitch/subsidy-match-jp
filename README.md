# Subsidy Match JP

中小企業向けの補助金マッチングMVPです。  
公式情報を確認した主要制度をカード化し、選択式の質問だけで候補を絞り込める構成にしています。

## 目的

- 補助金情報が多すぎて比較しづらい問題を解消する
- 申請可否の前に「どの制度が近いか」を短時間で見つける
- 公式サイトへの導線と最終確認日を明示して、誤認を減らす

## 実装内容

- Vite + React + TypeScript の静的サイト
- 用途、従業員規模、投資規模、所在地、受付状況の選択式フィルタ
- スコアリングによる候補表示
- 各制度カードに補助額、補助率、申請期間、注意点、公式URL、根拠ソース、最終確認日を表示
- 地域別の公式窓口導線

## 収録制度

- デジタル化・AI導入補助金2026
- ものづくり補助金 第23次
- 小規模事業者持続化補助金＜一般型・通常枠＞ 第19回
- 小規模事業者持続化補助金＜創業型＞ 第3回
- 新事業進出補助金 第3回
- 事業承継・M&A補助金 十四次公募
- 中小企業省力化投資補助金 カタログ注文型
- 中小企業省力化投資補助金 一般型 第6回

## 起動

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## 注意

このMVPは「候補を見つける」ことを目的にしています。  
最終的な申請可否は必ず最新の公募要領と公式事務局で確認してください。
